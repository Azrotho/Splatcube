# Splatcube

Splatcube est un plugin Minecraft qui est une adaptation de Splatoon dans Minecraft

## Installation

Pour installer le plugin, il faut le mettre dans le dossier plugins de votre serveur Minecraft ainsi que NickAPI *(Disponible dans les libs)*

## Usage

Pour utiliser le plugin, il faut faire la commande /debug help
